# Deno-API

Simple API using deno js

# End Points
"/books" (GET) - returns all books stored in database\
"/books/:id" (GET) - returns particular book based on book id\
"/books" (POST) - accept book details based on the model\
"/books/:id" (DELETE) - deletes the book stored in database based on id\
